Page Design
    Home Page
  1. The purpose of this page is that for whenever a users enters the site they should enter here and be able to easily naviagte around the site
  2. The audience is for everyone, customer, client, admin. It should be very user friendly
  3. The content on this page is to give an introduction of the site, and let the user navigate freely
  4. The user will not have to enter data on this page
  5. This page will include buttons to other parts of the page, and links to other parts of the page/ other pages
    About
  1. The purpose of this page is to introduce my client to their consumers as to be more connected to the client base
  2. The audience is mainly for the customer
  3. The content on this page is focused on describiing my client to the intended users
  4. The user will not have to enter any data on this page
  5. The page will include links in the nav but no other links
    Portfolio
  1. The purpose of this page is to show case all of my clients work on one page
  2. The audience is once again for the customer so that they can see how impressive the work is
  3. The content on this page will display all the work from my client, and have cool animations of them loading in
  4. The user will not have to enter any information on this page
  5. The page will also not have any links or buttons other than the links in the nav bar
    Quote
  1. The purpose of this page is to allow users to get a quote on art they want personalized
  2. The audience is once again for the customer
  3. The content on this page will be a form
  4. The user will have to enter txet data
  5. There will be buttons to submit and reset the form
    Products
  1. The purpose of this page is to allow users to view what the client may be selling
  2. The audience is my client and the customers he is trying to reach
  3. The content on this page will display photos and text information about each photo, with buttons and links
  4. There will be buttons and links to social media pages or where ever my client wants to sell their products. Whether that be facebook marketplace or ebay
