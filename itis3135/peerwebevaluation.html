Peer 1: Will, Poole

  • C- Contrast: This website has very good contrast with a lighter background than the foreground and unique element design. The size and shape of the elements are all different, bringing attention to them in order of importance.
  • R- Repetition: This website utilize repetition well, each section is similar to its respective group. Therefore the user never feels lost at any point.
  • A- Alignment: This website leaves some to be desired on alignment, all of the images are aligned to the left, some images are next to each other, and many there is not a consistent margin
  • P- Proximity: Elements that are similar are together but most items are not consistent in proximity, items feel scattered and the user can get kind of lost
